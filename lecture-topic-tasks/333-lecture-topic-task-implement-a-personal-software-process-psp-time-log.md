## üìù Description
The Personal Software Process (PSP) is a method for engineers to improve their planning and quality by relying on "personally collected data". A fundamental component of this is keeping a detailed "time log on all activities". This task involves creating and maintaining this log, specifically recording fields like Date, Start, Stop, Interruption, Delta (net time), and Activity . This will provide a practical application of PSP and create a personal dataset for improving time estimates.
